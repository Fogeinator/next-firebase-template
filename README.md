## Getting Started

First, add the firebase config variables in the `.env` file.

Next, rename the file `.env` to `.env.local`

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
